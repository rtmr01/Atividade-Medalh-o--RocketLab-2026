# Atividade-Medalh-o--RocketLab-2026
Pipeline de dados ponta a ponta no Databricks (PySpark) seguindo a arquitetura Medallion (Bronze, Silver e Gold). Processamento do dataset Olist com ingestão de arquivos Delta, consumo de API do Banco Central para câmbio USD/BRL, deduplicação por Window Functions e orquestração via Databricks Jobs (Workflow).
